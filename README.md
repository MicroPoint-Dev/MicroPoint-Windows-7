# MicroPoint-Windows-7
PowerPoint 交互复刻版 Windows 7 (Build 7601 SP1 RTM)

> 这是一个基于 PowerPoint 制作的交互式界面复刻项目，旨在还原 Windows 7 操作系统的视觉风格与基础交互体验。

## 项目目标
- 还原 Windows 7 主题（Aero Glass）的视觉风格
- 复刻经典开始菜单、任务栏、窗口边框、图标样式
- 实现基础系统交互（如窗口最小化/最大化、开始菜单弹出等）

## 如何使用
1. 下载最新的 Release 包
2. 在 PowerPoint 中打开"Setup.ppsm"（建议 Office 2019/2021/365，不支持 WPS Office）
3. 启用宏（如果需要交互功能）
4. 全屏放映即可体验

## 版本说明
- 当前版本：v6.1.7601.1000
- 复刻对象：Windows 7 SP1 RTM (Build 7601)
